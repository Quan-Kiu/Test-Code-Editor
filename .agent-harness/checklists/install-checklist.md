# Install Checklist

- [ ] Run install dry-run first.
- [ ] Review files that would be copied.
- [ ] Confirm existing files are not overwritten accidentally.
- [ ] Merge only after dry-run is acceptable.
- [ ] Read runtime entrypoint after merge.
- [ ] Fill first project docs: project brief, requirements, product, stories.
- [ ] Fill validation commands from actual package scripts.
- [ ] Run project validator if available.
- [ ] Record skipped files and why they were preserved.
