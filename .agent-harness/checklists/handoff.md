# Handoff Checklist

Use when moving from GPT sandbox to local, local to preview, or preview to public release.

- [ ] Current route is named.
- [ ] Current route status is recorded.
- [ ] Sandbox/local evidence status is recorded.
- [ ] Validation commands and results are listed.
- [ ] Browser/gameplay evidence is linked or summarized.
- [ ] Evidence ledger or validation report is updated.
- [ ] External/enduser readiness is separate from route status.
- [ ] Downstream blockers are listed.
- [ ] Next route is named.
- [ ] Local continuation steps are listed.
- [ ] No public/enduser readiness is claimed from sandbox-only evidence.
