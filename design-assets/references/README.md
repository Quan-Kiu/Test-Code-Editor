# references

Place approved design image files for this category here and register each source-of-truth image in `design-assets/manifest.json`.

Keep large editable source files outside this folder unless the project explicitly allows them. Export stable PNG/WebP/SVG references for agents, QA, and docs.

## Quality use

Reference assets are inspiration only unless approved. Record source/license status and ensure Visual/Composition and License/Compliance roles can distinguish reference-only from source-of-truth.

Reference review must cite `docs/role-gate-quality-map.md` when distinguishing Visual/Composition, License/Compliance, Product, and Accessibility concerns.
