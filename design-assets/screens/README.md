# screens

Place approved design image files for this category here and register each source-of-truth image in `design-assets/manifest.json`.

Keep large editable source files outside this folder unless the project explicitly allows them. Export stable PNG/WebP/SVG references for agents, QA, and docs.

## Quality use

Screen assets should show whole-frame composition, spatial balance, focal safe zone, and dominance. Browser screenshots must be compared against these references during composition QA.
