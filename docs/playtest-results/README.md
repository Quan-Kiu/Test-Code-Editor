# Human Playtest Results Directory

Place completed anonymous session sheets here only after real sessions are conducted.

## File naming

- `PT-WB-001-session-<anonymous-code>.md` — Rescue comprehension session.
- `PT-WB-002-session-<anonymous-code>.md` — Standard full co-op run.
- `PT-WB-003-session-<anonymous-code>.md` — Reduced effects comparison.
- `PT-WB-issues-<date>.md` — Consolidated issue triage summary.

## Do not store here

Names, email addresses, participant photos/video/audio, consent forms, device serial numbers or any credential/private key.

A blank directory indicates that human playtesting has not yet occurred.
